Support for hot reloading internal assets.

Internal assets (e.g. shaders) are bundled directly into an application and can’t be hot reloaded using the conventional API.

Structs

DebugAssetApp
A helper App used for hot reloading internal assets, which are compiled-in to Bevy plugins.
DebugAssetAppRun
A label describing the system that runs DebugAssetApp.
DebugAssetServerPlugin
Facilitates the creation of a “debug asset app”, whose sole responsibility is hot reloading assets that are “internal” / compiled-in to Bevy Plugins.
HandleMap
A collection that maps internal assets in a DebugAssetApp’s asset server to their mirrors in the main App.
